• Sistema de música para discord.js-v12.
 
• Comandos:
 
play [p, tocar]
pause [pausar]
nowplaying [np]
resume [resumir]
skip [pular]
volume [vol]
stop [parar]
queue [q]

ytsr
ytdl-core
@discordjs/opus
